🗂 Quick Reference Cards

<table>
The outer container. Every other tag lives inside this.
<caption>
Optional title shown above (or below) the table.
<colgroup>
Groups columns so you can style a whole column at once.
<col>
One <col> per column inside <colgroup>. Sets width / bg.
<thead>
Wraps the header rows. Browsers keep it visible on long scrolls.
<th>
A header cell — bold & centred by default. Lives in <tr>.
<tbody>
Wraps all the main data rows of the table.
<tr>
One horizontal row. Put <th> or <td> cells inside it.
<td>
A regular data cell. The most common cell type.
<tfoot>
Wraps footer rows (totals, notes). Renders last visually.

🧠 Memory Tricks

  • <table> = the body — everything is inside it
  • <caption> = the name tag on a shirt
  • <thead> / <tbody> / <tfoot> = Head 🧠 · Belly 🫁 · Feet 🦶 of the table
  • <tr> = Table Row — a shelf on a bookcase
  • <th> = Table Header — the label on the shelf
  • <td> = Table Data — the actual book on the shelf
  • <colgroup> + <col> = painting vertical stripes on the bookcase
