Real-Time Monitoring (Interactive Tools) 
These tools refresh automatically to show live changes in CPU and memory usage. 
top: The standard, built-in real-time task manager. It sorts processes by CPU usage by default.
htop: A more user-friendly and colorful alternative to top that supports mouse interaction and scrolling. You may need to install it first (e.g., sudo apt install htop).
atop: An advanced monitor that focuses on long-term performance analysis and historical logging.
